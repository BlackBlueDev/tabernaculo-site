# Tabernaculo de Davi

Site institucional estatico criado com:

- `index.html`
- `style.css`
- `script.js`

## Como editar

1. Abra `index.html`.
2. Procure pelos comentarios `TROCAR ...`.
3. Atualize logo, imagens, nome do pastor, endereco, link do Google Maps, redes sociais, contatos e lideres dos departamentos.

## Como publicar no Netlify

1. Entre no painel do Netlify.
2. Escolha `Add new site` e depois `Deploy manually`.
3. Arraste a pasta do projeto inteira ou conecte o repositorio.
4. Se conectar o repositorio, publique a pasta raiz do projeto.

## Estrutura

- `index.html`: estrutura e conteudo do site
- `style.css`: visual, responsividade e animacoes
- `script.js`: menu mobile, header e animacoes ao rolar
